# 2×claude — Claude March 2026 Usage Promo Dashboard

> A minimal single-page dashboard that tells users **right now** whether they're in Claude's 2× off-peak usage window — with a live contextual countdown.

---

## What is this?

From **March 13–28, 2026**, Anthropic is doubling usage limits for Claude users during off-peak hours.

## Features

- **Live status card** — instantly shows if you're in the 2× zone, peak hours, or a weekend
- **Contextual countdown** — dynamically shows:
  - *2× active* → time until next peak window starts (so you know how long you have)
  - *Peak hours* → time until 2× off-peak kicks in
  - *Weekend* → time until 2× resumes on Monday
- **Card state transitions** — border color + top glow shifts green (active) / red (peak) / grey (weekend)
- **Promo-end countdown** — separate D:H:M:S timer counting down to March 28, 11:59 PM PT
- **Peak zone reference table** — ET / PT / GMT windows at a glance
- **Eligible plans grid** — Free, Pro, Max, Team (Enterprise excluded)
- **Where it applies** — all Claude surfaces listed as chips
- **FAQ accordion** — 5 common questions with toggle answers
- **Sticky nav** — with live blinking dot
- **Noise grain texture** — subtle aesthetic overlay via inline SVG filter
- **Fully responsive** — works on mobile, tablet, desktop
